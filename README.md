# Dialogue System Demo

This repository contains a playable build of the Dialogue System prototype.

## Run

Download the repository and launch:

DialogueSystemDemo.exe

## Controls

Next dialogue
Keyboard: Space
Gamepad: A

Skip dialogue
Keyboard: Tab
Gamepad: X

Replay
Click the Replay button

## Source Code

The full Unity project can be found here:
https://github.com/myrmytzi/unity-dialogue-system

Unity Version: 6.3
